Project Description :-

  Conducted a comprehensive end-to-end analysis of pizza sales data using SQL to derive actionable business insights. The project involved aggregating and transforming raw transactional data through joins, 
  grouping, CTEs, and advanced SQL techniques such as window functions. Key focus areas included calculating daily and cumulative revenue, identifying top-performing pizza categories and products, and uncovering sales trends over time.

Key Highlights :-

  Calculated and visualized revenue trends, including cumulative revenue over time, to understand business growth.
  
  Identified top 3 pizza types by revenue for each category using ranking functions like DENSE_RANK().
  
  Analyzed category-wise performance to pinpoint revenue drivers and underperforming segments.
  
  Utilized SQL window functions for efficient trend computation and ranking across dimensions.
  
  Built insights that can support strategic decision-making and menu optimization.

Tools & Technologies :-

  SQL (Advanced Queries, Window Functions, Aggregations, Joins, Subqueries).
